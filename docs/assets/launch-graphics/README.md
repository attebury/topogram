# Launch Graphics

These are the first polished static graphics for Topogram's alpha story.

They are derived from the Mermaid diagrams in:

- [docs/visual-explanation-path.md](../../visual-explanation-path.md)

Working rule:

- Mermaid diagrams remain the canonical source of truth for wording and structure.
- These SVGs are the polished launch-facing variants for pages, decks, screenshots, and evaluator support materials.

Current set:

- [hero-wedge.svg](./hero-wedge.svg)
- [change-boundary.svg](./change-boundary.svg)
- [brownfield-reconcile-flow.svg](./brownfield-reconcile-flow.svg)

Style rules for future additions:

- keep the visual language restrained and proof-oriented
- prefer horizontal flows and simple cards
- avoid framework logos and decorative architecture clutter
- do not introduce claims that do not already exist in the Mermaid source diagrams
