# Issues Apps

Runnable generated runtimes for the Issues example belong here.

Current generated outputs:

- `backend/`
- `web/`
- `web-sveltekit/`
- `local-stack/`

For this example:

- `web/` is the generated Vite + React Router app
- `web-sveltekit/` is the generated SvelteKit app
- `local-stack/` is the SQLite-first runnable bundle built around the React default
